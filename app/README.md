# Friend's App Directory

This directory is for your friend's application that will be accessible via `app.mutinai.com`.

## Instructions for Your Friend

1. **Clone the repository:**
   ```bash
   git clone https://github.com/MutinAI/mutinai-site.git
   cd mutinai-site
   ```

2. **Add your app files to this `app/` directory**

3. **Commit and push your changes:**
   ```bash
   git add .
   git commit -m "Added my app"
   git push origin master
   ```

## Deployment

Your app can be deployed from this directory to `app.mutinai.com` as a subdomain.

Replace this README with your app's documentation once you add your files!
